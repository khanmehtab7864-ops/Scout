# Security Policy

## Supported Versions

Currently, only the latest development builds and official releases of **PathScout** are supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of PathScout seriously. If you find a security vulnerability or a severe bug that could compromise user systems, please do **NOT** open a public issue. 

Instead, please report it responsibly by:
1. Opening a **Draft Security Advisory** directly on the GitHub repository under the "Security" tab.
2. Alternatively, you can contact the maintainer directly via email at: `khanmehtab7864@gmail.com`

Please include a detailed description of the issue, steps to reproduce it, and any relevant logs or code snippets. We will acknowledge your report within 48 hours and work on a fix.
